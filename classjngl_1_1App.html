Detailed Description

There can only be one instance of this class which should be created before the window is shown.

Example:

#include <jngl/App.hpp>
#include <jngl/main.hpp>
JNGL_MAIN_BEGIN {
jngl::App app("Awesome Game Name");
// ...
app.mainLoop();
} JNGL_MAIN_END
