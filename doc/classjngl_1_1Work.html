◆ onQuitEvent()

void onQuitEvent ( )
virtual

Gets called when the user closes the main window or quit() has been called.

On Android this will happen when the back button is pressend, on iOS when the app is closed or put in the background.

To continue with the main loop, call cancelQuit().
